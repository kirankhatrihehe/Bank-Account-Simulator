# Checkpoint 3: Third Feature Plan

## Feature Name
View Account Details

## Description
This feature allows the user to view the details of a specific account, including account number, owner name, and current balance.Users enter an account number and the program displays the information.

## Why This Feature?
This feature builds on the first two features (Deposit and Withdraw) by letting users verify account information and see updated balances after transactions.

## Classes Involved
- Account (uses getters for accountNumber, ownerName, balance)
- Main program (for menu and user input)

## Expected User Workflow
1. User selects "View Account Details" from the menu.
2. User enters the account number.
3. Program searches for the account.
4. If account exists, details are displayed; otherwise, an error message appears.
